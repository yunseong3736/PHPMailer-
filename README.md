PHP Mailer 네이버 가이드라인
===========================

## 목표
PHP Mailer 라이브러리를 활용하여 네이버 SMTP 서버를 제어한다.  

----------------------------------------------------------------------------

> ## PHP Mailer 라이브러리 설치
> Composer (dependency manager)를 이용하여 추가  
> Composer 관련 파일은 깃헙에 올리지 않음
----------------------------------------------------------------------------
> ## 주의 사항  
> 네이버 2단계 인증에서 애플리케이션 비밀번호를 생성하여 사용하되 잘 기록할 것.
